# Zenn CLI

* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

## New Content

### 新しい記事を作成する

```bash
bun run new:article
```

### 新しい本を作成する

```bash
bun run new:book
```

### 投稿をプレビューする

```bash
bun run preview
```

## Update CLI

```bash
bun add zenn-cli@latest
```
